# Bunco Java Game

A console-based Bunco dice game implemented in Java.

## Description
This project simulates the Bunco dice game where a player competes against the computer. 
The game follows the traditional Bunco rules with rounds, dice rolls, and score tracking.

## Features
- Player vs Computer gameplay
- Dice rolling simulation
- Score tracking
- Object-oriented design using Java classes

## Technologies
- Java
- Object-Oriented Programming (OOP)

## How to Run
1. Clone the repository
2. Compile the program
3. Run the main class

Example:

javac BuncoGame.java  
java BuncoGame
